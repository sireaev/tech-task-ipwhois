# Tech Task - ipwhois lookup
`The REST API for ipwhois, with caching the query for period of time using cron.`

### Requirements
- pnpm (recommended)
- node.js


## Install
```
pnpm i
pnpm run db:create
```

## Run dev mode
```
pnpm start
```

## Other alternatives
- using redis/memcached
